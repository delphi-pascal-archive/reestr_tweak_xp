Шрифты

Использовать кодовую страницу 1251 вместо 1252

Данная опция решает проблемы с отображением русских шрифтов в некоторых программах. XP Tweaker меняет два параметра "1250" и "1252". В некоторых редких случаях можно исправить другие параметры.

HKLM\SYSTEM\CurrentControlSet\Control\Nls\CodePage

STRING 1250, значение "c_1251.nls" - опция включена, "c_1250.nls" - выключена
STRING 1252, значение "c_1251.nls" - опция включена, "c_1252.nls" - выключена

Переключение на русскую раскладку в DOS-приложениях *

В некоторых DOS-программах невозможно переключить раскладку на русский язык, данная опция поможет вам решить эту проблему. Переключение на русскую раскладку происходит по нажатию Ctrl+Right Shift, на английскую - Ctrl+Left Shift. Чтобы изменения вступили в силу необходимо перезагрузиться.

HKLM\SYSTEM\CurrentControlSet\Control\Keyboard Layout\DosKeybCodes

STRING 00000409, "ru" - опция включена, "us" - выключена

Если опция включена, то XP Tweaker также добавляет строку

"lh %SystemRoot%\system32\kb16.com ru" в %SYSTEMROOT%\SYSTEM32\autoexec.nt, если выключена, удаляет её.

Примечание: при загрузке XP Tweaker для проверки включена опция или нет, использует только параметр STRING 00000409 (т.е. не ищет строку в autoexec.nt). При записи в autoexec.nt XP Tweaker сначала проверит, существует ли там строка "lh %SystemRoot%\system32\kb16.com ru" и если она уже существует, то запись в файл не производится.

Зависшие приложения

Интервал, по истечении которого не отвечающее приложение считается зависшим:

По истечении этого времени Windows XP считает, что приложение зависло. По умолчанию значение равно 5 000 миллисекунд.

HKCU\Control Panel\Desktop

STRING HungAppTimeout, значение - время интервала в миллисекундах. XP Tweaker позволяет выставить значение в диапазоне от 1 до 20 секунд.

 

Ссылки по теме:

Windows 2000 Resourse Kits (HungAppTimeout)

Завершать зависшее приложение

Интервал, в течение которого Windows XP ожидает, перед закрытием приложения. Меньшее значение ускоряет процесс завершения работы, но это может привести к тому, что будут закрываться не зависшие приложения. По умолчанию интервал равен 20 000.

HKCU\Control Panel\Desktop

STRING WaitToKillAppTimeout, значение - время интервала в миллисекундах. XP Tweaker позволяет выставить значения от 1 до 30 секунд

 

Ссылки по теме:

Windows 2000 Resourse Kits (WaitToKillAppTimeout)

Завершать зависший сервис

Интервал, в течение которого Windows XP ожидает, перед закрытием зависшего сервиса. Меньшее значение ускоряет процесс завершения работы, но это может привести к тому, что будут закрываться не зависшие приложения. По умолчанию интервал равен 20 000.

HKLM\SYSTEM\CurrentControlSet\Control

STRING WaitToKillServiceTimeout, значение - время интервала в миллисекундах. XP Tweaker позволяет выставить значения от 1 до 60 секунд

Автоматически завершать зависшие приложения

Эта опция используется для автоматического закрытия без всякого предупреждения всех зависших программ. Удобно при завершении работы компьютера, если приложение не отвечает, то не надо нажимать на кнопку, чтобы его завершить.

HKCU\Control Panel\Desktop

STRING AutoEndTasks, значение 1 - автоматически завершать зависшие приложения, 0 - ждать ввода пользователя

Проверка файлов

Проверка файлов

Выполняет проверку и восстановление поврежденных системных файлов, возможно при этом потребуется компакт-диск с дистрибутивом Windows XP. XP Tweaker выполняет команду sfc.exe /scannow. Параметр /scannow - немедленная проверка без выдачи запроса, вы также можете использовать другие параметры, например, /scanonce - разовая проверка при следующей загрузке системы, для более подробной информации запустите в командной строке sfc без параметров. Если программа sfc находит, что защищенный файл был переопределен, подходящая версия файла восстанавливается из папки %SYSTEMROOT%\system32\dllcache или если таковой нет, с компакт-диска. Более подробную информацию о защите файлов Windows XP вы можете найти в справочной системе.

Путь к папке временных файлов

Путь к папке временных файлов

Некоторые старые программы не поддерживают длинные пути с пробелами. Рекомендуется использовать путь в стиле DOS 8.3, например, с:\windows\temp. Также этот параметр можно настроить в Свойствах системы - Дополнительно, кнопка Переменные среды.

HKCU\Environment

EXPAND STRING TMP, значение - путь к папке временных файлов
EXPAND STRING TEMP, значение - путь к папке временных файлов

AVI файлы

Корректное удаление avi файлов

В Windows XP часто возникают проблемы с удалением AVI файлов, система сообщает, что файл занят программой. С помощью данной опции можно устранить эту проблему.

Включение опции - XP Tweaker выполняет следующую команду:

regsvr32 /u %SystemRoot%\system32\shmedia.dll

Выключение опции (галочка снята)

regsvr32 %SystemRoot%\system32\shmedia.dll

 

Состояние опции программа проверяет с помощью содержимого ключа

HKLM\SOFTWARE\Classes\CLSID\{87D62D94-71B3-4b9a-9489-5FE6850DC73E}\InProcServer32

Примечание: При включении опции, вкладка "Сводка" в свойствах AVI файла будет отсутствовать. Не будет работать предварительный просмотр в Проводнике.
