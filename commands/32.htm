Скрытие/Добавление иконок на Рабочем столе

Для скрытия/добавления иконок на рабочем столе, используется следующий раздел реестра:

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\HideDesktopIcons\ ClassicStartMenu

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\HideDesktopIcons\ NewStartPanel

DWORD {208D2C60-3AEA-1069-A2D7-08002B30309D}, значение 1 - скрыть значок Сетевое окружение, 0 - показывать на рабочем столе

DWORD {871C5380-42A0-1069-A2EA-08002B30309D}, значение 1 - скрыть значок Internet Explorer, 0 - показывать на рабочем столе

DWORD {20D04FE0-3AEA-1069-A2D8-08002B30309D}, значение 1 - скрыть значок Мой компьютер, 0 - показывать на рабочем столе

DWORD {645FF040-5081-101B-9F08-00AA002F954E}, значение 1 - скрыть значок Корзины, 0 - показывать на рабочем столе

Примечание: если параметр не существует, значок Корзины показывается

DWORD {450D8FBA-AD25-11D0-98A8-0800361B1103}, значение 1 - скрыть значок Мои документы, 0 - показывать на рабочем столе
