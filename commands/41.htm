XP Tweaker - Internet Explorer
Автоматически подгонять размеры рисунка

Internet Explorer автоматически подгоняет рисунки так, чтобы они полностью умещались на экране и полосы прокрутки не появлялись. При необходимости можно отключить данную возможность.

HKCU\Software\Microsoft\Internet Explorer\Main

STRING Enable AutoImageResize, значение "yes" - автоматическое изменение размера изображений включено
Запретить автоматические обновления IE

HKCU\Software\Microsoft\Internet Explorer\Main

DWORD NoUpdateCheck, значение 1 - запрещает автоматические обновления
Скрывать редко используемые пункты в Избранном

В меню показываются только часто вызываемые пункты меню, для доступа к редко вызываемым пунктам, необходимо нажать на специальную стрелочку. Чтобы изменения вступили в силу, необходимо перезапустить Internet Explorer

HKCU\Software\Microsoft\Internet Explorer\Main

STRING FavIntelliMenus, значение "yes" - скрывать редко используемые пункты
Удалить пароль на ограничение доступа к сайтам

Удаляет пароль на ограничение доступа к сайтам по содержанию. Данный пароль можно установить в Свойствах обозревателя, вкладка Содержание, раздел Ограничение доступа.

HKLM\Software\Microsoft\Windows\CurrentVersion\policies\Ratings

BINARY Key, значение - зашифрованный пароль и параметры ограничения. Удалите параметр, если хотите удалить пароль
Не выводить окно "Подключение удаленного доступа" при автономной работе

Internet Explorer при автономной работе выводит окно "Подключение удаленного доступа" если информация не доступна локально. В большинстве случаев это не важная информация (рекламные баннеры) и подключаться к Интернету нет необходимости. На больших страницах Internet Explorer может выдать окно несколько раз, прежде чем полностью отобразит информацию. С помощью данной опции можно запретить IE выводить окно подключения.

HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings

DWORD EnableAutodial, значение 1 - разрешить вывод окна, 0 - запретить
Открывать ярлыки в новом окне IE

Internet Explorer при открытии ссылки заменяет содержимое последнего открытого окна, с помощью данной опции можно настроить IE так, чтобы ярлыки открывались в новом окне. (Требуется перезапуск Internet Explorer).

HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main

DWORD AllowWindowReuse, значение 1 - открывать в последнем открытом окне, 0 - открывать в новом окне
Запретить фоновые звуки

Опция запрещает Internet Explorer проигрывать фоновые звуки на web-страницах.

HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main

STRING Play_Background_Sounds, значение "yes" - разрешить проигрывать фоновые звуки, "no" - запретить
Путь к папке закачек по умолчанию

Данная папка будет всегда появляться первой для сохранения загружаемых файлов.

HKCU\Software\Microsoft\Internet Explorer

STRING Download Directory, значение путь к папке закачек
Редактор для просмотра исходного кода HTML страниц

Редактор, вызываемый через пункт меню Вид -> Просмотр HTML- кода.

HKLM\SOFTWARE\Microsoft\Internet Explorer\View Source Editor\Editor Name

STRING по умолчанию путь к редактору.
Поиск из панели адресов

Параметр влияет на поведение Internet Explorer, если он не находит страницу.

HKCU\Software\Microsoft\Internet Explorer\Main

DWORD AutoSearch, Internet Explorer 6.0 использует следующие значения:

0 - Не выполнять поиск из адресной строки
6 - Показать результаты и перейти на самый похожий узел
5 - Только перейти на самый похожий узел
4 - Только показать результаты в главном окне

Использовать автозаполнение для

Веб-адресов

Включить/выключить автозаполнение в строке адреса.

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\AutoComplete

STRING AutoSuggest, значение "yes" - автозаполнение веб-адресов используется
Автозаполнение форм

Включить/отключить автозаполнение форм. Internet Explorer выводит подсказки, в которых можно выбрать ранее введенные данные.

HKCU\Software\Microsoft\Internet Explorer\Main

STRING Use FormSuggest, значение "yes" - автозаполнение форм используется
Автозаполнение имен пользователей и паролей в формах

HKCU\Software\Microsoft\Internet Explorer\Main

STRING FormSuggest Passwords, значение "yes" - автозаполнение имен пользователей и паролей в формах используется

 
Запрос на сохранение пароля

HKCU\Software\Microsoft\Internet Explorer\Main

STRING FormSuggest PW Ask, значение "yes" - Internet Explorer выводит окно с сообщением сохранять пароль или нет
