XP Tweaker - Windows Media Player
Включить MP3 кодирование

Включение встроенного кодека для копирования музыки с компакт диска в формате *.mp3 (MPEG-1 Layer 3)

HKLM\SOFTWARE\Microsoft\MediaPlayer\Settings\MP3Encoding

Максимальный битрейт:
DWORD HighRate, XP Tweaker выставляет значение 192000, что равно 192 Кбит, хотя, скорее всего вам удастся кодировать с максимальным битрейтом 128 Кбит.

Минимальный битрейт:
DWORD LowRate, XP Tweaker выставляет значение 56000, что равно 56 Кбит

Средний-высокий битрейт
DWORD MediumHighRate, XP Tweaker выставляет значение 128000, что равно 128 Кбит - оптимальное значение

Средний битрейт
DWORD MediumRate, XP Tweaker выставляет значение 64000, что равно 64 Кбит

Если опция выключена, XPTweaker удаляет ключ HKLM\SOFTWARE\Microsoft\MediaPlayer\Settings\MP3Encoding полностью, при загрузке проверяется, существует ли данный ключ, если да, то опция включена.

Примечание: Если в Windows XP установлен Service Pack 1, то необходимо установить внешний кодек.

Автоматически загружать кодеки из Интернета

Если на компьютере нет необходимого видео или аудио кодека, то Windows Media Player попытается найти его в Интернете и затем загрузить его. Если опция выключена, Media Player сначала спросит, загрузить кодек или нет, а если включена, кодек будет загружен и установлен без выдачи сообщения.

HKCU\SOFTWARE\Microsoft\MediaPlayer\Preferences

DWORD UpgradeCodecPrompt, значение 1 - выдавать сообщение перед загрузкой кодека, 0 - загружать автоматически

Примечание: WMP8 использует однобайтовый двоичный тип данных (hex:00 или hex:01), но вполне корректно работает с dword.

Автоматические обновления

HKCU\SOFTWARE\Microsoft\MediaPlayer\Preferences

DWORD UpgradeCheckFrequency, может принимать следующие значения:

    • 0 - Один раз день
    • 1 - Один раз в неделю
    • 2 - Один раз в месяц

Примечание: WMP8 использует двоичный тип данных, но вполне корректно работает с dword. (соответственно hex:00,00,00,00, hex:01,00,00,00, hex:02,00,00,00)
