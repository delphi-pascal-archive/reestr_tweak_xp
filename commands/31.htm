XP Tweaker - Корзина
Название корзины

Изменяет имя корзины. Обновите рабочий стол, чтобы изменения вступили в силу.

HKEY_CLASSES_ROOT\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}

STRING по умолчанию, значение - имя корзины
STRING LocalizedString, значение - имя корзины
Добавить значок корзины в Мой компьютер

Добавляет значок корзины в Мой компьютер. Обновите Проводник (нажмите F5), чтобы изменения вступили в силу.

В разделе
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\ MyComputer\NameSpace, необходимо создать ключ {645FF040-5081-101B-9F08-00AA002F954E}, чтобы удалить значок корзины - удалите ключ

Контекстное меню корзины

Удаление и добавление новых пунктов в контекстное меню корзины.

Для удаления пункта меню, удалите соответствующий ключ в разделе:

HKEY_CLASSES_ROOT\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}\Shell

Чтобы добавить новое меню, создайте новый ключ. Значение STRING по умолчанию - название пункта меню. Далее создайте подключ Command, значение STRING по умолчанию которого - исполняемая строка.

Пример добавления блокнота в контекстное меню программы:

HKEY_CLASSES_ROOT\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}\Shell\MyNotepad

STRING по умолчанию = "Блокнот"

HKEY_CLASSES_ROOT\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}\Shell\MyNotepad

STRING по умолчанию = "notepad.exe" (или полный путь "c:\windows\notepad.exe", измените его, если каталог Windows отличается от приведенного)
