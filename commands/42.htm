XP Tweaker - Локальные сети
Не сканировать сеть на наличие сетевых принтеров

Windows XP по умолчанию каждый раз сканирует сеть на наличие сетевых принтеров и назначенных заданий. С помощью данной опции можно отключить данную возможность, тем самым, повысив доступ к компьютерам в сети.

При включении опции XP Tweaker удаляет ключ:

HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\RemoteComputer\ NameSpace\{2227A280-3AEA-1069-A2DE-08002B30309D}

при отключении опции восстанавливает его.
Не сканировать сеть на наличие назначенных заданий (Scheduled Tasks)

Описание см. выше. Используется следующий ключ:

HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\RemoteComputer\ NameSpace\{D6277990-4C6A-11CF-8D87-00AA0060F5BF}
