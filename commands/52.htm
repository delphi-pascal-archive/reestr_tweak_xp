XP Tweaker - MSN Messenger

MSN Messenger

Удалить встроенный Windows Messenger

Данная опция относится только к встроенному Windows Messenger, если вы ставили более новую версию, то для ее удаления используйте "Установку и удаление программ" в Панели управления. При открытии Outlook запускает встроенную версию Windows Messenger, даже если установлена более новая версия, используйте данную опцию чтобы удалить встроенный Windows Messenger из системы.

Если опция включена, XP Tweaker изменяет следующие параметры в реестре:

HKLM\SOFTWARE\Policies\Microsoft\Messenger\Client

DWORD PreventAutoRun, значение 1 - запрет автоматического запуска
DWORD PreventAutoUpdate, значение 1 - запрет обновлений
DWORD PreventBackgroundDownload, значение 1 - запрет загрузки файлов
DWORD PreventRun, значение 1 - запрет запуска

Данные значения никак не влияют на удаление, а само удаление происходит так:

XP Tweaker выполняет команду rundll32 advpack.dll,LaunchINFSection %SYSTEMROOT%\inf\msmsgs.inf,BLC.Remove

Примечание: При запуске, XP Tweaker читает значение DWORD PreventRun, если оно равно 1, то считается, что Windows Messenger уже удален и опция становиться не доступной.

Опция не доступна для Windows Server 2003 (нет Windows Messenger)

Если в Windows XP установлен пакет обновлений Servise Pack 1, то при выполнении команды произойдет ошибка error unregistering the OCX 16422, игнорируйте ее, Windows Messenger удалится без проблем. Можно также использовать Панель управления - Установка и удаление программ для удаления Windows Messenger (только в XPSP1).
