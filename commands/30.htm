XP Tweaker - Настройка Рабочего стола
Версия Windows на Рабочем столе *

Выводит версию Windows в правом нижнем углу Рабочего стола

HKCU\Control Panel\Desktop

DWORD PaintDesktopVersion, значение 1 - выводить версию Windows на Рабочий стол, 0 - не выводить.

Запретить рабочий стол *

Скрывает иконки с рабочего стола и делает не доступным контекстное меню.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD NoDesktop, значение 1 - запрещает рабочий стол
Запретить Active Desktop

Запрещает установку рабочего стола в виде веб-страницы.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD NoActiveDesktop, значение 1 - запрещает Active Desktop
Скрыть вкладки "Тема" и "Оформление" в свойствах экрана

Запретить изменение тем и оформления окон Windows XP

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System

DWORD NoDispAppearancePage, значение 1 - скрыть вкладки
Скрыть вкладку "Рабочий стол"

Скрывает вкладку Рабочий стол в свойствах экрана

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System

DWORD NoDispBackgroundPage, значение 1 - скрыть вкладку
Скрыть вкладку "Заставка"

Скрывает вкладку Заставка в свойствах экрана

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System

DWORD NoDispScrSavPage, значение 1 - скрыть вкладку
Скрыть вкладку "Параметры"

Скрывает вкладку Параметры в свойствах экрана

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\System

DWORD NoDispSettingsPage, значение 1 - скрыть вкладку
