XP Tweaker - Виртуальный диск
Создание и удаление виртуального диска

Данная утилита предназначена для создания новых виртуальных дисков на компьютере. С помощью виртуального диска можно быстро осуществить доступ к любой папке на компьютере или в сети.

Создание дисков происходит с помощью программы subst.exe:
subst [имя диска] [путь к папке]

Удаление виртуального диска:
subst [имя диска] /D
Создавать виртуальный диск при загрузке компьютера

Создавать виртуальный диск при загрузке компьютера

HKCU\Software\Microsoft\Windows\CurrentVersion\Run

STRING VirtualDrive-<Drive:> - значение subst <Drive:> <Path>

Например: VirtualDrive-B: - "subst B: D:\Мои документы\XP Tweaker\XPTweaker\hlp"
Создавать виртуальный диск для всех пользователей компьютера

Создавать виртуальный диск для всех пользователей компьютера

HKLM\Software\Microsoft\Windows\CurrentVersion\Run

STRING VirtualDrive-<Drive:> - значение subst <Drive:> <Path>

Например: VirtualDrive-B: - "subst B: D:\Мои документы\XP Tweaker\XPTweaker\hlp"
