Убрать имя пользователя из меню Пуск *

Убирает имя пользователя из меню Пуск (только если меню в стиле XP). Чтобы изменения вступили в силу, необходимо перезагрузиться.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD NoUserNameInStartMenu, значение 1 - убирает имя пользователя, 0 - показывать имя пользователя

Удалить пункт меню Выход из системы (Logoff) *

Запретить выход из системы из меню Пуск. Чтобы изменения вступили в силу, необходимо перезагрузиться.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD StartMenuLogoff, значение 1 - удаляет кнопку из меню

Удалить список часто используемых программ *

Не путайте данный параметр с "Недавно использовавшимися документами". Данный параметр убирает часто использующиеся программы, которые появляются в меню пуск, в левом столбце (только в стиле Windows XP). Чтобы изменения вступили в силу, необходимо перезагрузиться.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD NoStartMenuMFUprogramsList, значение 1 - удаляет список

Наверх
Не вести историю открытия документов

Не вести историю "Недавно использовавшихся документов". Windows XP создает в папке Documents and Settings\%USERNAME%\Recent ярлыки к запускаемым программам. Вы можете отключить данную возможность, если не пользуетесь ими.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD NoRecentDocsHistory, значение 1 - история открытия документов не ведется

Наверх
Очистка списка недавно использовавшихся документов

Очитка списка происходит во время выхода из системы (logout) или перезагрузки компьютера.

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

DWORD ClearRecentDocsOnExit, значение 1 - список очищается

Раскрывать меню при наведении и задержки указателя

Если опция включена, меню автоматически раскрывается наведении и задержки указателя, иначе необходимо щелкнуть левой кнопкой мыши для его открытия.

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

DWORD Start_AutoCascade, значение 1 - раскрывать меню автоматически, 0 - только после щелчка мыши.

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

DWORD Start_NotifyNewApps, значение 1 - раскрывать меню автоматически, 0 - только после щелчка мыши.

Элементы меню Пуск

Меню в стиле Windows XP

Примечание: переключатель "Меню в стиле XP" и "Классический вид меню" типа никак не влияет на меню. В зависимости от выбранного параметра вам будет предложено настроить опции для соответствующих меню. Оба меню используют различные параметры в реестре (см. ниже), исключением составляет параметр StartMenuAdminTools, он используется обоими типами меню, но меню в стиле XP использует тип DWORD, а классическое меню строковый тип. В зависимости от выбранного положения переключателя, параметр будет сохраняться, соответственно, с типом DWORD для меню в стиле XP и строковым типом для классического стиля.

Существует две возможности скрывать/показывать меню, а именно два ключа
HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced и
HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer или такой же ключ в HKEY_LOCAL_MACHINE
первый ключ просто скрывает/показывает пункты меню и их легко исправить стандартными средствами Windows XP, второй же ключ запрещает/разрешает показ меню, изменение параметров в этом ключе имеет больший приоритет и не изменяется стандартными средствами Windows XP. (см. раздел Защита - Меню Пуск)

Администрирование

Отображать в меню "Пуск"

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

DWORD Start_AdminToolsRoot, значение 0 - не отображать пункт меню, 1 - отображать как ссылку, 2 - отображать как меню

Отображать в меню "Все программы"

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

DWORD StartMenuAdminTools, значение 0 - не отображать пункт меню, 1 - отображать

Примечание: Windows XP не позволяет выбрать значение так, чтобы пункт меню Администрирование отображался в главном меню и не отображался в меню "Все программы", но вы можете это сделать с помощью XP Tweaker.

Показывать/скрыть пункт меню избранного Internet Explorer. Его удобно использовать, если вы работаете с Internet Explorer, если вы работаете с другим браузером, который использует свою папку избранного, этот пункт меню зачастую теряет смысл. Классический вид меню

Классический вид меню

Параметры меню классического типа сохраняются в строковом виде. Значение "YES" означает, что опция включена. Другие значения игнорируются (т.е. считается, что опция выключена). Для выключения опции рекомендуется ставить параметр "NO".

Примечание: в данном случае значения параметров "YES", "Yes" и "yes" - разные, используйте только "YES" чтобы опция вступила в силу.

Задержка выпадения меню *

Задержка выпадения меню *

Влияет на скорость показа меню Пуск и скорость выделения объектов в проводнике, если используется одинарный щелчок. По умолчанию задержка равна 400 миллисекунд. Чтобы изменения вступили в силу, необходимо перезагрузиться.

HKCU\Control Panel\Desktop

STRING MenuShowDelay, значение в миллисекундах, например "400". Обратите внимание, параметр имеет строковое значение, а не DWORD.

Примечание: в Windows 2003 Server значение "0" приводит к высокой нагрузке процессора при обращении к меню. XP Tweaker использует значение "1" вместо "0".
